# ENV-sensor-esp32

Consists off the following sensors:

- ESP32-S3
- SCD30 (CO2)
- SEN55 (Dust)
- MS5611 (Pressure)
- SSD1327 (Screen)
- BH1750 (LUX)

| Sensor   |   V   |  mA | SDA | SCL |
|----------|-------|-----|-----|-----|
| ESP32-S3 |       |     |     |     |
| SCD30    |   5   | 75  |     |     |
| SEN55    |   5   | 110 |     |     |
| MS5611   |  3.3  | >1  |     |     |
| SSD1327  |  3.3  | ~50 |     |     |
| BH1750   |   5   | >1  |     |     |
